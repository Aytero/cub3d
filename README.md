# cub3D
Maze-game made with a custom raycasting engine. Inspired by Wolfenstein 3D. 
\
\
\
**To launch: make && ./cub3D maps/map.cub**

## Movement:
WASD, Up, Down to move\
Left, Right to rotate\
Esc to exit\
